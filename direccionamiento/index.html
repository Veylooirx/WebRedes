Direccion IP

La dirección IP es el identificador del dispositivo dentro de una red y debe ser único dentro de los límites de dicha red.
La IP es un número que identifica de manera lógica y jerárquica a una interfaz de un dispositivo
El uso, formato, tipos y demás características del direccionamiento IP están incluidos en lo que se conoce como protocolo IP (Internet Protocol).
